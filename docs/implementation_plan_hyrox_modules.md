# 攀岩/赛事小程序功能模块架构方案 (基于最新结构表)

## 1. 首页核心栏目 (Home - Level 1)
*   **1.1 广告位 (Ad Slot)**: 顶部 Banner 轮播，用于核心赛事或岩馆推广。
*   **1.2 岩馆搜索 (Gym Search)**: 支持名称、区域、等级等维度的检索。
*   **1.3 岩馆展示 (Gym Display)**: 精选岩馆或附近岩馆的卡片式入口。
*   **1.4 赛事 (Events)**: 核心赛事板块入口。
*   **1.5 正在举办赛事活动报名 (Ongoing Registration)**: 快速报名通道入口。
*   **1.6 个人注册 (User Profile/Reg)**: 用户注册与身份管理入口。
*   **1.7 联系我们 (Contact Us)**: 客服及平台信息。
*   **1.8 多语种切换 (Multi-language)**: 全局语言环境切换（中/英等）。

---

## 2. 次级功能详情 (Secondary - Level 2)

### 2.1 岩馆详情与线路 (对应 1.3)
*   **岩馆介绍**: 岩馆简介、营业时间、设施描述。
*   **线路展示**: 按难度、类型展示岩馆内现有线路列表。

### 2.2 赛事分类列表 (对应 1.4)
*   **赛事一 / 二 / 三 / 四**: 按照类型、区域或时间维度划分的赛事列表。

### 2.3 个人信息管理 (对应 1.6)
*   **基础资料**: 昵称设置、联系方式（手机号/微信）绑定。
*   **付款功能**: 集成支付接口，支持报名费用支付。

---

## 3. 三级深度功能 (Tertiary - Level 3)

### 3.1 岩馆互动与评价 (对应 2.1)
*   **岩馆评价**: 用户评分与文字反馈。
*   **完攀打卡**: 标记个人完成的线路。
*   **线路评价**: 针对特定线路的难度/趣味性评价。
*   **视频记录**: 上传并查看线路评价及完攀过程视频。

### 3.2 赛事深度互动 (对应 2.2)
*   **信息资讯**: 赛事介绍、新闻发布、历史回顾、精彩精彩视频汇总。
*   **报名系统**: 详细报名信息查看、在线报名流程。
*   **赛事直播与进度**: 赛事实时进程追踪、多赛场/多机位实时直播。
*   **排名与回放**: 实时名次展示、与名次对应的选手比赛视频回放。

### 3.3 注册验证 (对应 2.3)
*   **身份验证**: 短信验证码或第三方实名校验。

## 4. 技术实施建议
*   **视频处理**: 需要稳定的 OSS 存储与视频点播（VOD）支持。
*   **实时性**: 排名与直播需要 WebSocket 或 SSE 支持。
*   **多语言**: 严格遵循 `i18n.js` 配置规范。
