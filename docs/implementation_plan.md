# 攀岩小程序产品优化升级计划

## 一、 目标描述
将现有的赛事报名系统升级为集场馆探索、社交互动（线路评价/视频）、高级赛事管理（抱石/速度赛规则）及多语言支持的综合性攀岩运动平台。

## 二、 核心功能模块

### 1. 攀岩馆生态
- **附近探索**: 基于 LBS 的场馆搜索与展示。
- **线路系统**: 线路图片管理、难度分级、完攀视频展示。
- **评价系统**: 用户上传短视频评价，展示完攀人信息。

### 2. 赛事引擎升级
- **抱石赛逻辑**: 定线员安装 -> 计时设备绑定 -> 自动计算多级别排名。
- **速度赛逻辑**: 认证赛道计时 -> 实时成绩更新 -> Top 8 名次公示。
- **直播中心**: 支持多赛道实时视频切换、实时名次滚动展示。

### 3. 用户与通知
- **国际化**: 中/韩/日/英/法/德/阿拉伯语切换。
- **提醒功能**: 消息订阅及报名、成绩提醒。

## 三、 方案确认
> [!NOTE]
> 1. **计时集成**: 目前不进行硬件对接，仅在后端记录及前端展示最终成绩。
> 2. **视频流**: 赛事详情页目前使用占位图/示意图，预留直播接口。
> 3. **多语言**: 仅实现 **中文** 与 **英文** 切换。
